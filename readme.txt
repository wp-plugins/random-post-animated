== Description ==

Displays randomly selected hyperlinks to posts on your site from a selection list according to an adjustable time interval. The selection list is compiled based on criteria that you specify in the widget settings.The widget can be added to the site footer or post/page sidebar.

**Key Features**

* Determine the maximum number of posts you would like the plugin to include in the selection list
* Include posts in the selection list according to category
* Exclude certain posts from the selection list
* Adjustable random selection animation/time interval speed
* Works with multiple widget instances
* Widget can be added to the site footer, post/page sidebar, or both

== Installation ==

**Plugin Directory**

1. Search the plugin directory for plugin.
2. Click 'Install Now'.
3. Activate the plugin.
4. The Random Post widget should now be visible in the 'Available Widgets' area in Appearance > Widgets


**Upload**

1. Download the plugin Zip file.
2. Go to plugins in the Wordpress Admin area.
3. Select 'Upload Plugin'
4. Activate the plugin.
5. The Random Post widget should now be visible in the 'Available Widgets' area in Appearance > Widgets
